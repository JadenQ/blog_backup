### Hexo的安装

https://hexo.io/zh-cn/

hexo server，本地调试

```
hexo server
```

补充，后期如果新增了文章，只需要执行 `hexo g -d` ，然后更新 Gitee Pages 服务即可

```
hexo g -d
```
